# App Test Images

This folder contains sample images used to test the **PinyinVision** app.

These images help verify that the app can:

- Detect Chinese characters
- Convert them to **pinyin**
- Provide **pronunciation**

The images are intentionally simple so that developers and users can easily test the core functionality of the app.

## Usage

You can test the app by:

1. Opening the **PinyinVision** app
2. Taking a photo of the image on your screen
3. Checking whether the app correctly detects the Chinese text and generates pinyin

These images are mainly for **development and testing purposes**.
